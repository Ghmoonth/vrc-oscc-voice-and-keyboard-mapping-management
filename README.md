# OSCC

OSCC 是一个面向 VRChat OSC 的 C#/.NET 桌面工具，包含键盘参数映射、语音参数控制和 AI 翻译模块。

## 映射部分
可以通过osc将键盘键位绑定到游戏内模型的参数，通过按键触发参数，比如更换衣物，打开特效，拿出物品，开启功能等
支持一个按键绑定多个参数

语音映射部分，自由选择输入音频的方式，音频接口，当识别声音与你所设定的意图相近或完全相等时，即可触发
例如口述亮度提升，脱掉帽子，即可自动实现功能开关，无需游戏内打开菜单


## 语音系列使用方法
自己找喜欢的AI平台注册api，拥有额度，填入api key即可使用（请谨慎保管好自己的api key）
deepseek仅能用于LLM，此步骤为文字间翻译部分
可只使用一个api即可（deepseek不支持多模态输入所以不行）
